## 구성요소

### 뉴스 크롤링

- 유명한 영어권 언론사의 헤드라인 뉴스 본문 크롤링

### LLM

- 크롤링한 뉴스 본문을 번역, 요약
- 뉴스 본문에서 중요 어휘를 요약

### 카카오톡 알림

- 가공된 정보를 사용자의 카카오톡 나와의 채팅으로 알림
